# OpertusMundi CLI

OpertusMundi client. 

## Quickstart

Copy configuration example files from `config-example/` into `src/main/resources/`, and edit to adjust to your needs.

`cp -r config-example/* src/main/resources/`

### Build

`mvn clean package`

### Run

`java -jar target/opertus-mundi-cli-1.0.0.jar`
